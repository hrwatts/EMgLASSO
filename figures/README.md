# Figures

This directory contains the figures used in the EMgLASSO manuscript.

## Regenerating Figures

To regenerate the manuscript figures exactly as stored here, run:

```powershell
python generate_figures.py
```

The example scripts in `examples/` remain useful for demonstrating package usage, but the committed manuscript figures are generated by the Python script above.

## Figure Descriptions

- **precision_networks_low_sparsity.png**: Network visualization of lower-sparsity precision matrices used in the manuscript.
- **precision_networks_high_sparsity.png**: Network visualization of higher-sparsity precision matrices used in the manuscript.

The script writes output directly into this directory.

## Requirements

- Python packages: `numpy`, `matplotlib`, `networkx`
- Repository root as the working directory so the script can write to `figures/`
